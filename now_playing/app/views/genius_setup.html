Genius Access Token Setup Guide

  1. First, create a Genius account. Genius tracks usage, so you might want to use one of your not-so-important email addresses for registration.
  2. Next, set up an API Client. On the New API Client form:
    • Provide an App Name which can be anything you want (e.g. "metadata-service").
    • For App Website URL, you can provide a random address which does not have to point to an existing website (e.g. "https://somerandomwebsite.org").
    • Leave the other fields blank.
  3. After submitting the form, you will be given the Client ID and Client Secret. What you need, however, is the Client Access Token. So click the "Generate Access Token" link.
  4. Finally, copy the generated Access Token and paste it into the "Genius Access Token" field of the plugin settings (under "Metadata Service"). Click "Save".

That's it! The metadata service should now be available when you select "Info View" from the menu of the Now Playing screen.

Note: The Access Token is given to you only once. If you lose it, you would have to go back to the API Client page and generate a new one.
