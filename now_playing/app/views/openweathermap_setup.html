OpenWeatherMap API Key Setup Guide

  1. To begin, sign up for an OpenWeatherMap account.
  2. Next, go to the "API keys" page. You will see that an API key has already been created for you.
  3. Copy the API Key and paste it into the "OpenWeatherMap API Key" field of the plugin settings (under "Weather Service"). Click "Save".

Finally, in the "Localization" section of the plugin settings, make sure you have provided the Geographic Coordinates of your location. With the coordinates and API key set, the plugin will be able to fetch weather information from OpenWeatherMap.
